## Development

After cloning the repo to your machine, run:

```shellscript
npm i
```

You only need to do it once, or if new packages were installed

Run the dev server:

```shellscript
npm run dev
```
## Important
Do not touch context.tsx, entry.client\entry.server.tsx, createEmotionCache.ts, and root.tsx
